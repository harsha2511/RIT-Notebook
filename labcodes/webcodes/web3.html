Apply CSS border property create the following table. Give proper caption for the table and do the following:-
a. Left Align the text, border spacing of 5px and cell padding of 15px
b. Use row span and Column span

<!DOCTYPE html>
<html>
<head>
<title>Table Example</title>
<style>
table {
border-collapse: separate;
border-spacing: 5px;
caption-side: top;
padding: 15px;
}
th, td {
border: 1px solid black;
padding: 15px;
text-align: left;
}
</style>
</head>
<body>
